This Code requires a folder called "data" that you will have to put the csv’s from the FCT software into

You will have to change the end date in the code to line up with the date of the most recent csv from the FCT software. The start date should always be the first day of the year that you are in

It will output an hourly averaged csv and a minute averaged csv for the year. The ASI does not take data overnight so there will be gaps in the data during these times.
